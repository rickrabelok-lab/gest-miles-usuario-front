# WS3 — Sentry (Implementation Plan)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans.

**Goal:** Observabilidade de erros de runtime no front (React) e no backend (BFF), reusando o projeto Sentry do manager, distinguido por `environment`/tag.

**Architecture:** Init **guardado por env** (sem DSN ⇒ no-op) → o código entra dormente e ativa quando `VITE_SENTRY_DSN` (front) / `SENTRY_DSN` (backend) forem setados no Vercel. **Tech Stack:** `@sentry/react` (front), `@sentry/node` (backend). Vitest (front), `node --test` (backend).

**Branch:** `feat/observabilidade-sentry`. `git fetch` + branch do `main`.

**Decisão do owner (2026-06-05):** **reusar o DSN do manager**; separar por `environment` (`usuario-front` / `usuario-backend`) + tag `app`. Owner seta os envs no Vercel; o código não muda conforme o DSN.

import * as Sentry from "@sentry/react";

let initialized = false;

/** Inicializa o Sentry só se houver DSN (no-op sem env). Reusa o projeto do manager,
 *  separado por environment/tag. */
export function initSentry() {
  const dsn = import.meta.env.VITE_SENTRY_DSN as string | undefined;
  if (!dsn || initialized) return;
  Sentry.init({
    dsn,
    environment: (import.meta.env.VITE_SENTRY_ENV as string | undefined) || "usuario-front",
    tracesSampleRate: 0.1,
    sendDefaultPii: false,
  });
  Sentry.setTag("app", "usuario-front");
  initialized = true;
}

export { Sentry };
```

import * as Sentry from "@sentry/node";

let initialized = false;

/** Inicializa o Sentry só se houver SENTRY_DSN (no-op sem env). Reusa o projeto do manager,
 *  separado por environment/tag. */
export function initSentry() {
  const dsn = process.env.SENTRY_DSN;
  if (!dsn || initialized) return;
  Sentry.init({
    dsn,
    environment: process.env.SENTRY_ENV || "usuario-backend",
    tracesSampleRate: 0.1,
  });
  Sentry.setTag("app", "usuario-backend");
  initialized = true;
}

/** Captura um erro no Sentry se inicializado. Best-effort: nunca lança. */
export function captureException(err) {
  try {
    if (initialized) Sentry.captureException(err);
  } catch {
    /* ignore */
  }
}
```
