Central de Notificações

  • 4 filtros — Todas · Vencendo · Saldo · Promos
  • Ponto roxo (•) por item não lido; badge vermelho no Bottom Nav enquanto houver pelo menos um
  • Marcar tudo lido — ✓ no header zera todos os dots e remove o badge
  • 3 prioridades de cor — Vermelho (urgente) · Amarelo (promo) · Cinza (info)
  • Tap numa notificação abre a tela relacionada (ex: vencendo → tela do programa)
  • Estado vazio limpo quando não há novas notificações

Dados necessários: tabela notifications no Supabase com campos: tipo, programa_id, corpo, lida, criado_em

Histórico de Emissões

  • 3 stats no topo — total de emissões, milhas usadas, valor estimado gerado
  • Timeline por mês — rota + programa + classe + milhas + status
  • 4 status visuais — Confirmado (verde) · Pendente (âmbar) · Realizado (cinza) · Cancelado (vermelho)
  • Tela de detalhe — localizador, data/hora do voo, taxa paga, valor gerado, assento
  • Filtro por ano / programa — para localizar emissões antigas
  • Estado vazio com CTA "Registrar emissão" quando não há histórico

Dados necessários: tabela emissoes — origem, destino, programa_id, milhas, taxa, status, localizador, data_voo
