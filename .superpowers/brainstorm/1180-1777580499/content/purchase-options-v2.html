🔧 Como foi estruturado no código

Uma constante GESTMILES_EMISSION_ENABLED = false em src/config/features.ts controla se o card da Gest Miles aparece ou não.

O componente já existe renderizado, só fica oculto enquanto a flag está false. Quando você for ativar: mude para true, pronto.

🚀 Para ativar no futuro

1. Abra src/config/features.ts
2. Mude GESTMILES_EMISSION_ENABLED = false para true
3. O card completo com "Emitir agora" aparece automaticamente na tela.

Nenhuma outra mudança necessária — o card já está pronto no JSX.
